# Patent Notice

## Provisional Patent Applications

This software implements technology covered by the following provisional patent application(s):

### Context Scaffolding System for Facilitating Agentic Software Development

**Filing Date**: October 28, 2025
**Status**: Provisional Patent Application Filed

**Technology Coverage**:
- Version-control-native memory layer that stores agent and human decision data inside version controlled repositories
- Context architecture through which a supervisory "principal" agent can communicate concepts and the location of their implementations to subordinate AI agents
- Real-time visual codebase map that renders repository structure, agent activity, and codebase structural analytics in an interactive spatial interface
- Machine-readable document association system linking documentation files with implementation files
- Living documentation with automatic flagging when source files change
- Temporal context replay for reconstruction of collaboration history
- Cross-repository context integration for organization-wide knowledge graphs

**Applicable Components**:
- `.alexandria/` directory structure and contextual memory system
- CodebaseView concept and implementation
- MemoryPalace class and associated APIs
- Machine-readable context mapping (YAML/JSON associations)
- Pre-commit hooks for documentation enforcement
- City Map visualization transforming file trees into spatial representations
- Rules engine for documentation quality validation
