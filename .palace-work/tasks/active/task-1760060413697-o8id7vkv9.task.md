# Request

## Request
Add an `updateTaskStatus(taskId: string, status: TaskStatus): void` method to the MemoryPalace class.

## Context
We have implemented a Palace Tasks API in the electron app that includes an UPDATE_TASK_STATUS handler at `src/main/palace-tasks/palaceTasksHandlers.ts:85`. However, the MemoryPalace class in @a24z/core-library@0.1.28 does not expose a method to update task status.

## Current State
- MemoryPalace has: `getTasks()`, `getTask()`, `deleteTask()`, but no `updateTaskStatus()`
- The TaskStore is private and cannot be accessed directly
- Our handler needs to update task status based on user actions in the UI

## Proposed API
```typescript
class MemoryPalace {
  // ... existing methods
  
  /**
   * Update the status of an existing task
   * @param taskId - The unique identifier of the task
   * @param status - The new status for the task
   * @returns boolean indicating success
   */
  updateTaskStatus(taskId: string, status: TaskStatus): boolean;
}
```

## Related Files
- Electron app handler: `src/main/palace-tasks/palaceTasksHandlers.ts`
- API interface: `src/shared/main-process-api-interfaces/PalaceTasksAPI.ts`

## Workaround
Currently returning false and logging a warning until this is implemented.
